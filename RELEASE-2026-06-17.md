# RdEnglish UI 重设计 — 2026-06-17

## 改动概览

### 配色升级
从单色灰度 → 双色品牌体系：

| | HEX | 浅色模式主色 | 深色模式主色 |
|---|---|---|---|
| **深棕灰** | `#312F2C` | 主色 (Primary) | 强调色 (Accent) |
| **薄荷绿** | `#ABD1C6` | 强调色 (Accent) | 主色 (Primary) |

浅色/深色模式自动交换角色确保对比度。

### 仪表盘重构
- **旧**：欢迎区 + 统计栏 + 写作模块 + 工具模块 + Pro 升级卡 的复杂网格
- **新**：两个居中大按钮——「开始写作」和「小工具」
  - 按钮交替配色：深棕灰文字+薄荷绿背景 / 薄荷绿文字+深棕灰背景
  - 悬停放大动画 + 阴影

### 新增子页面
- **`/start-writing`**：翻译 / 优化 / 批改 三卡片网格（交替配色）
- **`/tools`**：笔记本 / 语法病历 / 学习报告 / 单词本 四卡片网格（交替配色）

### 导航栏精简
- 删除左侧 logo 和「英语写作助手」文字
- 删除所有功能名称前的图标（✨ 优化 → 优化，📚 单词本 → 单词本）
- 桌面端：分组竖线分隔（核心工具 | 学习工具 | 社交+记录）
- 移动端：分区标题（核心工具 / 学习工具 / 社交 / 更多）

### 返回按钮
